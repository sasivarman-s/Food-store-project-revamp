## 🌟 Overview

**Food Store** is a sleek, single-page restaurant/food-ordering website designed to give visitors a smooth, appetizing browsing experience — from the hero banner all the way to checkout-style calls to action. It's built entirely with **HTML** and **Tailwind CSS**, focusing on clean UI, responsive layouts, and real-world front-end practices.

> 🔥 **[Click here to view the live website](https://sasivarman-s.github.io/Food-store-project-revamp/)**

---

## ✨ Features

- 🏠 **Eye-catching Hero Section** — "Get Delicious Food at Anytime" with quick actions to *View Menu* or *Order Now*
- 🍽️ **Food Service Highlights** — Showcases the fine-dining, at-home delivery experience
- 📋 **Explore Menu** — Categorized food cards (Non-Veg Starters, Grilled Seafood, Hyderabadi Biryani, Soups, and more) with imagery
- 🥗 **Fresh & Organic Section** — Highlights healthy, chemical-free food options
- 🚚 **Delivery & Payment Info** — Multiple payment options, live order tracking, and online payment discounts
- 🙌 **Customer Appreciation Section** — Rewards/redeem section for loyal customers
- 📱 **Fully Responsive Design** — Built with Tailwind CSS utility classes for a seamless experience across devices
- 🔗 **Social & Contact Footer** — Instagram, Gmail, X (Twitter), email, and address details

## 📂 Getting Started (Run Locally)

1. **Clone the repository**
   ```bash
   git clone https://github.com/sasivarman-s/Food-store-project-revamp.git
   ```
2. **Navigate into the project folder**
   ```bash
   cd Food-store-project-revamp
   ```
3. **Open `index.html`** in your browser — that's it! No build step required.

   *(If you're using the Tailwind CLI/build process instead of the CDN, run `npm install` and your Tailwind build command as configured in the project.)*
